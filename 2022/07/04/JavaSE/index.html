1）public class与class的区别？

如果声明为public class，则类名和文件名必须一致。如果声明为class，则类名和文件名可以不一致。

在一个文件中，声明为public class的最多有1个。但是声明为class的，可以有多个。定义多少个class，就会生成多少个字节码文件（.class）。

1）数据溢出问题

整数的默认类型是int。如果想要特指某个数据类型，则数值的范围不能超过该类型的范围。

如果非要用小范围去存储大范围的数据，则涉及到了数据类型转换问题。

数据类型的转换

基本数据类型转换有两种：一种叫自动类型转换，一种叫强制类型转换。

1）自动类型转：大范围去接收小范围的数据。

Int类型可以接收byte、short、char。

Long类型可以接收byte、short、int、char

Float类型可以接收byte、short、int、long、char

double类型可以接收byte、short、int、long、float、char

注意：byte、short、char之间不能转换

public > protected > default > private

笔试题：重载和重写的区别

区别点:

区别点 重载 重写
单词 overload override
定义 方法名相同，参数个数、类型、顺序不同 方法名、返回值类型、参数列表完全不同
修饰符 可以有不同的访问修饰符 被子类重写的方法不能拥有比父类更加严格的权限
返回值类型 可以改变返回值类型 不能改变
范围 发生在一个类中 发生在继承类
异常 可以抛出不同的异常 重写方法一般抛出新的检查异常或者重写的方法名更加宽广的检查异常
构造方法 可以被重载 不能被重写
静态方法 可以被重载 如果父类声明的是静态方法，则子类也必须是静态方法，同时考虑访问权限的问题

    public void fun() {

    }

    public B() {
        super();
    }
    public void fun() {

    }

笔试题：this 和 super 的区别

区别点 this super
属性 调用本类中的属性，如果本类中没有，则调用父类中非私属性 调用父类中的属性
方法 调用本类中的方法，如果本类中没有，则调用父类中非私方法 调用父类中的方法
构造方法 调用本类中的构造方法，必须放在构造方法的首行 调用本父类中的构造方法，必须放在子类构造方法的首行
特殊性 表示当前对象 无此概念

final关键字

final关键字可以修饰属性、方法、类

  • 使用final声明的类不能又子类

  • 使用final的方法不能被重写

  • 使用final声明的变量为常量，常量不可以修改，常量名一律大写

定义全局常量：

public static final int PI = 3.14
