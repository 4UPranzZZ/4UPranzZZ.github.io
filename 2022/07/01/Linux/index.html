1.ls命令

显示文件和目录列表（list）

常用参数：
-l（long） 长格式显示文件和目录信息
-a（all） 显示所有文件和目录，包括隐藏文件和目录
-R 递归显示指定目录下的文件清单，即会显示指定目录分支内各子目录中的文件清单

-la 查看文件和目录组合使用命令

3.操作文件命令

1.创建空文件：touch 文件名

[root@localhostghr ~]# touch a.txt
[root@localhostghr ~]# ls
anaconda-ks.cfg  a.txt  initial-setup-ks.cfg
[root@localhostghr ~]# ls -l
总用量 8
-rw-------. 1 root root 1888 9月  23 10:58 anaconda-ks.cfg
-rw-r--r--. 1 root root    0 1月  14 11:26 a.txt
-rw-r--r--. 1 root root 1936 9月  23 10:58 initial-setup-ks.cfg
[root@localhostghr ~]#

1.如果1处是d，代表是目录

2.touch 代表摸一下，如果touch 一个已存在的文件，例如touch bb.txt，则不会新建，而是修改bb.txt的创建时间

2.创建目录（make directory）：mkdir

​ -p 父目录不存在情况下先生成父目录（parents）

3.复制文件或目录（copy）

​ cp 文件名 目录 复制文件到指定目录中

​ cp -r 目录名 目录 复制指定目录到指定目录

​ -r 递归处理 将指定目录下的文件和子目录一并拷贝

rm -rf 支持通配删除

rm -rf *.txt

配置linux系统中网络

1.查看当前ip地址
ip a 查看  默认网卡是否在ip地址
ens33 没有ip地址

2.编辑网络配置在每次启动时自动获取ip地址
1）配置当前vmware软件网络配置位 NAT模式
2）开启centos自动ip获取
    vi /etc/sysconfig/network-scripps/ifcfg-ens33
   修改默认配置
       ONBOOT=no
   改为
       ONBOOT=yes
3.加载配置生效
1）重启当前linux
2）重启网络服务  network
    systemctl restart network
4.测试访问外部
    ping www.baidu.com
